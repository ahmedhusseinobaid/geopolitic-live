## 🚀 النشر على Netlify (مضمون 100% — أخبار لحظية)

### الخطوة 1 — ارفع الملفات على GitHub
تأكد أن مستودعك يحتوي على:
```
├── index.html
├── netlify.toml
└── netlify/
    └── functions/
        └── news.js
```

### الخطوة 2 — أنشئ حساب Netlify
1. اذهب إلى **https://netlify.com**
2. اضغط **Sign up** → سجّل بحساب GitHub
3. اضغط **Add new site** → **Import an existing project**
4. اختر مستودع `geopolitic-live`
5. اضغط **Deploy site** ✅

### الخطوة 3 — تم! 🎉
سيكون موقعك على:
`https://your-site-name.netlify.app`
